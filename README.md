# Internet Programming Lab Experiments

This repository contains 14 web development experiments covering HTML, CSS, JavaScript, and React.

## Experiments

1. **HTML with Hyperlinks, Images and Tables** - Basic HTML elements demonstration
2. **HTML with Lists, Frames and Multimedia** - Lists, iframes, video and audio elements
3. **HTML Form with All Form Elements** - Complete form with all input types
4. **HTML with Hyperlinks, Lists and Frames** - Navigation and content organization
5. **Inline and Internal CSS** - CSS styling methods demonstration
6. **Internal and External CSS** - Multiple CSS implementation approaches
7. **JavaScript Current Date and Time** - Real-time clock display
8. **JavaScript Arrow Functions** - ES6 arrow function examples
9. **Form Validation with JavaScript** - Client-side form validation
10. **React State and Props** - React component state management
11. **CSS Font, Text and Background Properties** - Advanced CSS styling
12. **CSS Color, Table and Lists** - CSS styling for tables and lists
13. **JavaScript Classes and Inheritance** - OOP concepts in JavaScript
14. **Simple Interest Calculator** - JavaScript function implementation

## How to Run

Simply open the `index.html` file from any experiment folder in your web browser. No server or build process required.

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6+)
- React (CDN)

## Notes

Each experiment is self-contained and can be run independently. All code is browser-compatible and requires no additional dependencies except for Experiment 10 (React) which uses CDN links.
